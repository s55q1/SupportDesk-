# مشروع تخرج: نظام دعم فني بسيط

## الهدف
مشروع دعم فني بسيط ومقسم إلى ملفات منطقية مع واجهتين: واجهة الموظف ولوحة الفني.

## الهيكلية
- `config/database.php` : إعداد اتصال قاعدة البيانات.
- `includes/auth.php` : تسجيل الدخول وصلاحيات المستخدمين.
- `includes/search.php` : منطق البحث الذكي، الحلول السريعة، واقتراحات الفني.
- `pages/login.php` : صفحة تسجيل الدخول.
- `pages/dashboard.php` : لوحة تحكم عامة بعد تسجيل الدخول.
- `pages/employee.php` : واجهة الموظف.
- `pages/technician.php` : لوحة الفني.
- `pages/admin.php` : إدارة المستخدمين وصلاحيات النظام.
- `pages/reports.php` : تقارير التذاكر.
- `pages/logout.php` : تسجيل الخروج.
- `database.sql` : إنشاء الجداول وبيانات تجريبية.
- `index.php` : صفحة البداية للتنقل إلى نظام تسجيل الدخول.

## كيفية التشغيل
1. ضع المشروع داخل مجلد خادم محلي مثل `htdocs` في XAMPP أو `www` في Wamp.
2. استورد `database.sql` في phpMyAdmin أو MySQL.
3. عدل بيانات الاتصال في `config/database.php` إذا لزم الأمر.
4. افتح المتصفح واذهب إلى:
   - `http://localhost/SupportDesk/index.php`

## ملاحظات الاختبار السريعة
- إذا لم يكن PHP مثبتًا، فإن `run-server.bat` الآن يفتح `index.html` حتى تحصل على عرض توضيحي فوري.
- لتعيين كلمة مرور معروفة للمسؤول للاختبار، قم بتشغيل الأمر من جذر المشروع (يتطلب PHP CLI وDB قيد التشغيل):

```powershell
php tools/reset_admin_password.php
```

ثم قم بتسجيل الدخول باستخدام اسم المستخدم `admin` وكلمة المرور `admin123` وقم بتغيير كلمة المرور من `pages/admin.php`.

## وصف العمل
- واجهة الموظف تعرض فئات العطل وتبحث في الحلول السريعة من قاعدة البيانات.
- إذا لم تفد الحلول، يمكن فتح تذكرة جديدة.
- لوحة الفني تعرض التذاكر المفتوحة وتقترح حلولاً مشابهة من الأرشيف.

## ملاحظات
- الكود مكتوب بشكل نظيف ومبسط لمناقشة المشروع.
- يمكنك إضافة صفحة إدارة الحلول أو إغلاق التذاكر لاحقاً كتحسين.
